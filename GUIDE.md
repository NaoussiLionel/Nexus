# Nexus Architect — User Guide

## What is Nexus Architect?
An AI-powered mind-mapping canvas for planning projects. Sketch out ideas, organize them with AI, and arrange everything visually.

## Layouts

| Layout | Icon | Behavior |
|--------|------|----------|
| Tree | Tree ↓ | Top-down hierarchy (default) |
| Root | Root → | Left-to-right outline |
| 2-Sided | 2-Sided ↔ | Balanced branches left and right |
| Star | Star ☆ | Radial burst from center |

Select a layout from the toolbar dropdown, then click **Arrange** to apply it.

## Depth Limit
The canvas shows up to 3 levels at a time. When expanding a deep branch, Nexus auto-focuses on it so you can dive deeper without clutter. Click **Show full map** to zoom back out.

## AI Architect (Sidebar)
- Describe your project in natural language
- Ask for changes, expansions, or restructuring
- The AI adapts to your methodology (WBS, Agile, Kanban, Waterfall, OKR, etc.)
- New nodes are positioned without disturbing your manual layout

### AI Actions
- **Expanding a node**: Click the sparkle icon on any node → AI generates sub-items
- **Elaborate**: Open a node and click "Elaborate with AI" for detailed notes
- **Chat**: Type or use suggestion chips to start

## Node Operations
- **Drag**: Click and drag any node to reposition it
- **Select**: Click a node to select it (opens details drawer)
- **Multi-select**: Hold Shift and click nodes, or Shift+drag a marquee box on the canvas
- **Multi-drag**: Drag one selected node to move all selected nodes together
- **Batch delete**: Select multiple nodes and press Delete
- **Add child**: Click + on the node toolbar
- **Collapse**: Click the toggle at the bottom of a node
- **Focus**: Click the crosshair to isolate a branch
- **Delete**: Click the trash icon

## Details Drawer
Click any node to open its details panel (slides from the left). Edit the title, add notes, elaborate with AI, and save. Click the X or press Escape to close.

## Toolbar
- **Undo** (Ctrl+Z) — step back through changes
- **Search** (magnifier icon) — find nodes by title or description; matching nodes glow brass
- **Arrange** — auto-position all nodes per the selected layout
- **Layout** dropdown — choose Tree, Root, 2-Sided, or Star
- **Fit** — zoom to show every node
- **Zoom** — in/out buttons
- **Export** — JSON or Markdown outline
- **Import** — load a `.json` project file
- **Clear** — wipe the project (tap twice to confirm)

## Shortcuts
- Enter — send AI message
- Shift+Enter — new line in AI input
- Click node — open drawer
- Drag node — reposition (or drag selected group)
- Shift+click node — toggle multi-select
- Shift+drag on canvas — box-select nodes
- Delete — remove selected node(s)
- Drag canvas background — pan
- Scroll — zoom
- Escape — close drawer or cancel AI confirmation

## Tips
- Start with a blank canvas or let the AI build the first map from your description
- Manually arrange nodes to reflect your mental model — AI won't bulldoze your layout
- Use the Focus feature to work on complex branches without distraction
- Export to Markdown for task lists or documentation
